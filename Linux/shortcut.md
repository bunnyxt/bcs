## 快捷键

### bash

| 快捷键 | 作用 |
| -- | -- |
| `Ctrl + L` | 清屏，相当于`clear` |
| `Ctrl + A` | 光标移动到**行首** |
| `Ctrl + E` | 光标移动到**行尾** |
| `Ctrl + U` | 从光标位置剪切到**行首** |
| `Ctrl + K` | 从光标位置剪切到**行尾** |
| `Ctrl + Y` | 粘贴`Ctrl + U`或`Ctrl + K`剪切的内容 |
| `Ctrl + R` | 在历史命令中搜索 |

ref: [Bash常用快捷键](https://www.cnblogs.com/zgngg/p/10274018.html)
